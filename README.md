# Artifactorys
Cada pasta correspode a um experimento executado em cenário industrial:
A pastas 'ExperimentoPontual1' e 'ExperimentoPontual2' referem-se a erros de delay de acesso a funcionalidades.
A pasta 'ExperimentoSérieTemporal1', refere-se a um erro de alto consumo de memória ao executar uma ação.
A pasta 'ExperimentoSérieTemporal2', refere-se a um erro de alto consumo de CPU ao executar uma ação.
