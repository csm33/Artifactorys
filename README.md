# Artifactorys
Cada pasta corresponde a um experimento específico executado em cenário industrial:
As pastas 'ExperimentoPontual1' e 'ExperimentoPontual2', referem-se a erros de delay de acesso a funcionalidades.
A pasta 'ExperimentoSérieTemporal1', refere-se a um erro de consumo de memória ao executar uma ação.
A pasta 'ExperimentoSérieTemporal2', refere-se a um erro de consumo de CPU ao executar uma ação.
